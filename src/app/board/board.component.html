Game of Life Simulator

How to use

Use the WASD keys to modify the tooltip. W and S to cycle back and forth through the options. D allows you to rotate the tooltip by 90 degrees. A reflects the tooltip horizontally.

Click Save to save the current game state as a new tooltip option. Delete will remove the currently selected tooltip. Step moves the simulation forward by one iteration.

Tips

You can use the tooltip to save the game state when you select a larger grid size.

The tooltip can kind of work like an eraser. It toggles the live or dead state of each highlighted cell when you click.
